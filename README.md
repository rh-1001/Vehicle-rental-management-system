# Vehicle Management System

A desktop Vehicle Management System built with **C++** and the **Qt Framework**.

## About

This project is being developed to strengthen my understanding of:

- Object-Oriented Programming (OOP)
- C++
- Qt Widgets
- GUI application development
- Software design principles

## Planned Features

- Add vehicles
- View all vehicles
- Search for vehicles
- Update vehicle information
- Delete vehicles
- Save and load vehicle data
- Input validation
- Vehicle statistics

## Technologies

- C++
- Qt Framework
- CMake
- Git
